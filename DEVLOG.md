# 개발일지 — YJ IT Solutions 회사 사이트

## 프로젝트 개요
- **목적:** IT 개발 회사 공식 홈페이지 제작
- **기술 스택:** React 19, Vite, React Router DOM
- **컬러 테마:** 다크블루 / 로열블루 (메인), 다크그린 / 다크레드 (포인트)
- **저장소:** https://github.com/yong275/rest02

## 2026-06-08

### 환경 세팅
- GitHub 리포지토리 클론 (`yong275/rest02`)
- Vite + React 프로젝트 초기화
- `react-router-dom` 설치 (페이지 라우팅)
- `gh-pages` 설치 (GitHub Pages 배포)

### 디렉토리 구조 설계
```
src/
  components/
    Header/       # 고정 헤더, 스크롤 감지, 모바일 햄버거 메뉴
    Footer/       # 4단 그리드 푸터
  pages/
    Home/         # 메인 페이지 (히어로, 통계, 서비스, WHY US, CTA)
    About/        # 회사소개, 핵심가치, 연혁 타임라인
    Services/     # 서비스 상세, 진행 프로세스
    Contact/      # 문의 폼
  styles/
    variables.css # CSS 변수 (색상, 폰트, 간격 등)
    global.css    # 전역 스타일, 공통 클래스
```
