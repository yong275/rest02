# YJ IT Solutions — 회사 공식 홈페이지

IT 전문 기업 회사 소개 사이트입니다.

## 📁 프로젝트 구조

```
src/
├── components/
│   ├── Header/       # 고정 헤더, 스크롤 감지, 모바일 햄버거 메뉴
│   └── Footer/       # 4단 그리드 푸터
├── pages/
│   ├── Home/         # 메인 페이지
│   ├── About/        # 회사소개, 핵심가치, 연혁
│   ├── Services/     # 서비스 상세
│   └── Contact/      # 문의 폼
└── styles/
    ├── variables.css # CSS 변수 (색상, 폰트 등)
    └── global.css    # 전역 스타일
```

---

## 📄 페이지 구성

- **홈 (`/`)** — 히어로, 통계, 서비스 카드, WHY US, CTA
- **회사소개 (`/about`)** — 회사 개요, 핵심 가치, 연혁 타임라인
- **서비스 (`/services`)** — 4개 서비스 상세, 진행 프로세스
- **문의 (`/contact`)** — 문의 폼
